Definition:
A slot of time on a schedule that may be available for booking appointments.

  id 1..1idDefinition:
The logical id of the resource, as used in the URL for the resource. Once assigned, this value never changes.


Comments:
The only time that a resource does not have an id is when it is being submitted to the server using a create operation.

  serviceType 0..*CodeableConceptBinding: Argonaut_Visit_Types (extensible)
Definition:
The type of appointments that can be booked into this slot (ideally this would be an identifiable service - which is at a location, rather than the location itself). If provided then this overrides the value provided on the availability resource.

  schedule 1..1Reference(Schedule)Definition:
The schedule resource that this slot defines an interval of status information.

  status 1..1codeBinding: Argonaut_Slot_Status_Codes (required)
Definition:
busy | free | busy-unavailable | entered-in-error.

  end 1..1instantDefinition:
